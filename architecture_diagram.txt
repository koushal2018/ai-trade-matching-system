User Flow:
1. User accesses the application through AWS Amplify hosted frontend
2. Frontend makes API calls to API Gateway
3. API Gateway routes requests to Lambda functions
4. Lambda functions process requests and interact with DynamoDB and S3
5. For document processing, Lambda triggers AI Vision services
6. Processed data is stored in DynamoDB
7. Reports are generated and stored in S3

Data Flow:
1. Trade documents uploaded to S3
2. AI Vision Lambda processes documents using Textract/Rekognition
3. Extracted trade data stored in DynamoDB
4. Reconciliation process matches trades in DynamoDB
5. Reports generated and stored in S3
6. Frontend displays data from DynamoDB and S3
